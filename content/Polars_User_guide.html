🌈 データフレームって何？

データフレームは2次元のデータ構造で、表みたいな感じなの！行と列にラベルがついてて、違うタイプのデータも一緒に扱えるの！マジ便利！分析とか加工とかが超らくちん！💕

  • 📈 RAMより大きいデータもへっちゃら！
  • 👌 使い方が一貫してて分かりやすい！
  • 🎯 データの型がちゃんとしてる！

🌟 実際の使い方

import polars as pl q = ( pl.scan_csv("docs/assets/data/iris.csv") .filter(pl.col("sepal_length") > 5) .group_by("species") .agg(pl.all().sum()) ) df = q.collect()
